# HACK HUSTLE

## About This Project

This is a modern web application built with React and TypeScript.

## Tech Stack

This application is built with modern web technologies:

- **Vite** - Fast build tool and dev server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Beautiful, accessible component library
- **Tailwind CSS** - Utility-first CSS framework

## Development

### Local Development

To work on this project locally:
```sh
# Clone the repository
git clone <your-git-url>

# Navigate to project directory
cd <project-name>

# Install dependencies
npm i

# Start development server
npm run dev
```

**Requirements**: Node.js and npm ([install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

### Other Options

- **GitHub Web Editor**: Click the "Edit" button on any file in the repository
- **GitHub Codespaces**: Click the "Code" button → "Codespaces" tab → "New codespace"
